Carry on!

If you've ever traveled by plane, you know that there are strict rules when it comes to carry-on luggage. A handbag has three dimensions: width, length and height. Some company set the following rule for the carry-on:

allowed = (height <= 10 < width <= 35 < length <= 40) or (height + width + length <= 80)
